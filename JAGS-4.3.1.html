Two binary packages are available for JAGS 4.2.0. If you are using R to interface with JAGS then you must ensure that you download the correct binary:


If you are using R 4.2.0 or later then install JAGS-4.3.1.exe

If you are using R 4.1.3 or earlier then install JAGS-4.3.0.exe


There is no difference in the JAGS library or modules between the two releases.
